ETL Pipeline: Data Processing with Pandas

📌 The Problem:

Corporate data often arrives in unstructured or mixed formats (.csv and .xlsx), containing null values, poorly formatted columns, and inconsistencies. Manual processing of these files causes operational bottlenecks and increases the risk of critical failures.


💡 The Solution:

This project consists of a Python script focused on automating the Extraction and Transformation (ETL) phase. The pipeline reads raw files, applies data cleansing rules, and standardizes the structure for subsequent loading into a relational database.


🛠 ️ Technologies Used

    - Python 3.12: Core language for automation.
    - Pandas: Main library for DataFrame manipulation and data cleaning.

🚀 How to Run

    - Clone this repository.
    - Ensure Pandas is installed (pip install pandas).
    - Place the raw data files into the data/ directory.
    - Run the script your_script_name.py.
